# Websfolio

Personal portfolio React app powered by Vite.

## Requirements

- Node.js 24.15.0 or newer
- npm 11.13.0 or newer

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

The local site normally opens at `http://localhost:5173`.

## Production Build

```bash
npm run build
```

The production files are generated in `dist`.

## Preview Build

```bash
npm run preview
```
